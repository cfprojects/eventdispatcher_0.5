EventDispatcher
Joe Rinehart (contact@firemoss.com)
EventDispatcher Alpha (0.5).

In ActionScript 3, the EventDispatcher is a core class that implements the "Subject"
side of the Observer pattern. Extending it allows you to create classes that can
dispatch events as well as have event listeners registered/removed/managed.

*** INSTALLATION ***

Place the CFCs in this installation into /com/firemoss/eventdispatcher (or use a mapping
to their location).

http://eventdispatcher.riaforge.org - The homepage for EventDispatcher.
